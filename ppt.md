## Slide 5: How It Works (Architecture)
**Logo**: [AI Tutor Logo]
**Content**: 
We use a **Multi-Agent closed-loop system**:
1. **The Architect**: Interviews the user to build a custom curriculum blueprint.
2. **The Tutor**: Delivers interactive lessons using Web Search, YouTube, and Image Intelligence.
3. **The Assessor**: A strict "Gatekeeper" that quizzes the user and only grants access to the next lesson upon proven mastery.
4. **The Synthesizer**: Generates spaced-repetition flashcards and verifies official certifications.
